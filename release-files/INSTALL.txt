Steam Input Controller Patch for InControl Games (macOS)

Supported games: Tricky Towers, Overcooked! 2

1. Open the DMG and run "Steam Input Controller Patch.app"
2. If macOS blocks the app, go to System Settings -> Privacy & Security and click Open Anyway
3. Patcher auto detects installed games in Steam. If multiple games are found, select which to patch. If the game is installed elsewhere, select the .app manually.
4. Click "Patch"
5. Enable Steam Input: Steam -> Right-click the game -> Properties -> Controller -> Enable Steam Input

To restore original: run the app again and click "Restore"

Note: If the game won't launch after patching, click "Codesign" to re-sign the app.

More info: https://github.com/fr-eed/tricky-towers-controller-fix
