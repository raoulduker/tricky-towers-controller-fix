Tricky Towers Controller Patch (macOS)

1. Open the DMG and run "Tricky Towers Controller Patch.app"
2. If macOS blocks the app, go to System Settings -> Privacy & Security and click Open Anyway
3. Patcher auto detects default Steam installation, if game is installed elsewhere select TrickyTowers.app
4. Click "Patch"
5. Enable Steam Input: Steam -> Tricky Towers -> Properties -> Controller -> Enable Steam Input

To restore original: run the app again and click "Restore"

More info: https://github.com/fr-eed/tricky-towers-controller-fix
